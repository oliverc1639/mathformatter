How to write integrals:

  • int x^2 dx → ∫ x² dx (preview shows pretty math)
  • int int x*y dx dy → double integral
  • int int int z dx dy dz → triple integral
  • int[0,1] x^2 dx → definite integral
  • int[0,inf] e^(-x) dx → improper (0 → ∞)
  • int_0^1 x^2 dx also works for definite integrals
  • Type inf in bounds to get ∞ automatically; solver still understands it.
  • Superscripts / subscripts (like x^2, a_1) are rendered nicely in the preview and steps.
