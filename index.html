Integral Calculator

Enter = Solve Shift+Enter for a newline.

Type an integral on its own line and press Enter. The answer will appear two lines below that line. Use Shift+Enter to insert a normal newline without solving.

Supported syntax (for now):
  • ∫ x^2 dx → indefinite integral w.r.t. x
  • ∫[0,1] x^2 dx → definite integral from 0 to 1
  • integrate(x^2, x) → Nerdamer syntax
  • defint(x^2, x, 0, 1) → definite integral (Nerdamer)
  • If you just type x^2, it assumes ∫ x^2 dx.
